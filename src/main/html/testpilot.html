TestPilot.js

Welcome to TestPilot, an interactive JavaScript test framework.

Load the code to be tested and TestPilot test classes by dropping JavaScript files anywhere on the page. If the files contain TestPilot test classes, the test classes will be loaded and the tests will be immediately invoked. If the files contain application code, the files will be loaded and will be accessible to TestPilot test classes.

Load the code to be tested (e.g.: application code) first, then load the TestPilot test classes. Click the Reload option in the Details section to reload both application code and TestPilot test classes.
